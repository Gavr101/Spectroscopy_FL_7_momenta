# Руководство по использованию методов моментного анализа

## Обзор

В этом руководстве описаны особенности использования реализованных классов для моментного анализа сигналов и изображений. Все классы наследуют от `BaseEstimator` и `TransformerMixin` из scikit-learn, что позволяет интегрировать их в ML пайплайны.

## Импорт классов

```python
from src import (
    MomentumTransformer1D, MomentumTransformer2D,
    LegendreTransformer1D, LegendreTransformer2D,
    FourierTransformer1D, FourierTransformer2D,
    ZernikeTransformer2D
)
```

## 1. Полиномиальные моменты (MomentumTransformer)

### Особенности
- Вычисляют статистические моменты сигнала (интеграл, среднее, дисперсия и т.д.)
- Для 2D: комбинации моментов по x и y
- Хорошо подходят для описания формы распределений
- Чувствительны к смещению и масштабу

### Когда использовать
- Анализ формы сигналов/распределений
- Детекция аномалий в данных
- Сравнение распределений по моментам
- Простые признаки для ML моделей

### Пример использования
```python
# 1D случай
trf = MomentumTransformer1D(order=3, x_bounds=(0, 10))
coeffs = trf.fit_transform(data)  # data.shape = (batch, N)

# 2D случай
trf = MomentumTransformer2D(order=2, x_bounds=(0,1), y_bounds=(0,1))
coeffs = trf.fit_transform(data)  # data.shape = (batch, N_y, N_x)
```

## 2. Полиномы Лежандра (LegendreTransformer)

### Особенности
- Ортонормированный базис на [-1,1]
- Оптимальны для гладких функций
- Быстрое затухание коэффициентов для гладких сигналов
- Хорошо сохраняют локальные особенности

### Когда использовать
- Анализ гладких сигналов
- Спектральный анализ в ограниченном интервале
- Приближение функций полиномами
- Когда важна ортогональность базиса

### Пример использования
```python
# 1D случай
trf = LegendreTransformer1D(order=10)
coeffs = trf.fit_transform(data)  # data на [-1,1]

# 2D случай
trf = LegendreTransformer2D(order=5)
coeffs = trf.fit_transform(data)  # data на [-1,1] x [-1,1]
```

## 3. Гармонические моменты (FourierTransformer)

### Особенности
- Базис из синусов и косинусов на [-1,1]
- Отлично подходят для периодических сигналов
- Коэффициенты показывают частотный состав
- Хорошо для сигналов с резкими изменениями

### Когда использовать
- Анализ частотного спектра
- Детекция периодичностей
- Фильтрация шумов
- Работа с периодическими данными

### Пример использования
```python
# 1D случай
trf = FourierTransformer1D(order=20)
coeffs = trf.fit_transform(data)  # data на [-1,1]

# 2D случай
trf = FourierTransformer2D(order=10)
coeffs = trf.fit_transform(data)  # data на [-1,1] x [-1,1]
```

## 4. Полиномы Цернике (ZernikeTransformer2D)

### Особенности
- Ортонормированный базис на единичном круге
- Учитывают радиальную и угловую симметрию
- Автоматическое определение центра масс
- Хорошо для круглых/радиально-симметричных объектов

### Когда использовать
- Анализ оптических аберраций
- Обработка изображений с круговой симметрией
- Биометрия (анализ радужки)
- Анализ волновых фронтов

### Пример использования
```python
trf = ZernikeTransformer2D(order=15, x_bounds=(0,10), y_bounds=(0,10))
coeffs = trf.fit_transform(data)  # data.shape = (batch, N_y, N_x)
# Центр определяется автоматически по центру масс
```

## Общие рекомендации

### Выбор порядка разложения
- Начните с малого порядка (2-5) для тестирования
- Увеличивайте постепенно до достижения нужной точности
- Следите за переобучением при высоких порядках

### Предобработка данных
- Масштабируйте данные к соответствующим интервалам:
  - Legendre/Fourier: [-1, 1]
  - Momentum/Zernike: произвольные границы
- Нормализуйте амплитуду если необходимо

### Интеграция в ML пайплайны
```python
from sklearn.pipeline import Pipeline
from sklearn.ensemble import RandomForestClassifier

pipeline = Pipeline([
    ('momentum', MomentumTransformer2D(order=3)),
    ('classifier', RandomForestClassifier())
])
```

### Визуализация и отладка
- Используйте `get_pd_table()` для просмотра коэффициентов
- Сравнивайте оригинал и реконструкцию для оценки качества
- Проверяйте сохранение основных характеристик сигнала

### Производительность
- Для больших данных используйте меньший порядок
- Рассмотрите параллельную обработку батчей
- Кэшируйте преобразования для повторного использования

### Типичные применения
- **Спектроскопия**: анализ форм линий поглощения/излучения
- **Обработка изображений**: извлечение признаков формы
- **Анализ сигналов**: декомпозиция временных рядов
- **Оптика**: анализ аберраций и волновых фронтов
