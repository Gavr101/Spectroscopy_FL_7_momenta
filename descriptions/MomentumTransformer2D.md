# MomentumTransformer2D

## Описание

Класс `MomentumTransformer2D` предназначен для вычисления полиномиальных моментов двумерных данных. Полиномиальные моменты представляют собой интегралы вида:

$$M_{i,j} = \int_{x_{\min}}^{x_{\max}} \int_{y_{\min}}^{y_{\max}} x^i y^j f(x, y) \, dx \, dy$$

где $f(x, y)$ - функция, описывающая данные, $i$ и $j$ - порядки моментов.

Моменты используются для компактного представления распределения данных, включая центр масс, дисперсию и более высокие статистические характеристики.

## Взятие интегралов

Интегралы вычисляются численно с использованием прямоугольного правила интегрирования на равномерной сетке.

### Шаги вычисления:

1. **Определение сетки**: Данные представлены на сетке $N_x \times N_y$ точек в пределах $[x_{\min}, x_{\max}] \times [y_{\min}, y_{\max}]$.

2. **Шаг интегрирования**: 
   $$dx = \frac{x_{\max} - x_{\min}}{N_x - 1}, \quad dy = \frac{y_{\max} - y_{\min}}{N_y - 1}$$

3. **Интегральный элемент**: $dA = dx \cdot dy$

4. **Вычисление момента**:
   Для момента порядка $(i, j)$:
   $$M_{i,j} = \sum_{k=1}^{N_x} \sum_{l=1}^{N_y} x_k^i y_l^j f(x_k, y_l) \cdot dA$$

   где $x_k = x_{\min} + (k-1) \cdot dx$, $y_l = y_{\min} + (l-1) \cdot dy$.

### Центральные моменты

Для моментов выше первого порядка используются центральные моменты относительно центра масс:

$$x_0 = \frac{M_{1,0}}{M_{0,0}}, \quad y_0 = \frac{M_{0,1}}{M_{0,0}}$$

Тогда центральный момент порядка $(i, j)$:
$$M_{i,j}^c = \int (x - x_0)^i (y - y_0)^j f(x, y) \, dx \, dy$$

В коде центральные моменты вычисляются как:
$$M_{i,j}^c = \left( \sum (x - x_0)^i (y - y_0)^j f(x, y) \cdot dA \right)^{1/(i+j)}$$

## Параметры класса

- `order`: Максимальный порядок моментов
- `x_order`, `y_order`: Максимальные порядки по x и y (если не указаны, равны order)
- `x_bounds`, `y_bounds`: Границы интегрирования

## Применение

Моменты используются в анализе изображений, распознавании образов и статистическом моделировании для извлечения характеристик распределений.
