# FourierTransformer2D

## Описание

Класс `FourierTransformer2D` выполняет двумерное разложение Фурье для данных на квадрате [-1, 1] × [-1, 1]. Функция представляется как сумма гармоник:

$$f(x, y) \approx \sum_{m=0}^{M} \sum_{n=0}^{N} \left[ a_{mn} \cos(m\pi x) \cos(n\pi y) + b_{mn} \cos(m\pi x) \sin(n\pi y) + c_{mn} \sin(m\pi x) \cos(n\pi y) + d_{mn} \sin(m\pi x) \sin(n\pi y) \right]$$

где $a_{mn}$, $b_{mn}$, $c_{mn}$, $d_{mn}$ - коэффициенты Фурье.

Это позволяет анализировать частотный состав двумерных сигналов.

## Взятие интегралов

Интегралы вычисляются на нормализованной области [-1, 1] × [-1, 1] с использованием прямоугольного правила.

### Шаги вычисления:

1. **Нормализация координат**: Аналогично Лежандру, данные масштабируются к [-1, 1].

2. **Шаг интегрирования**: $dx' = dy' = \frac{2}{N-1}$

3. **Интегральный элемент**: $dA = \frac{4}{(N_x-1)(N_y-1)}$

4. **Вычисление коэффициентов** для каждой комбинации (m, n):

   - $a_{mn} = \int_{-1}^{1} \int_{-1}^{1} f(x', y') \cos(m\pi x') \cos(n\pi y') \, dx' \, dy'$
   
   - $b_{mn} = \int_{-1}^{1} \int_{-1}^{1} f(x', y') \cos(m\pi x') \sin(n\pi y') \, dx' \, dy'$ (если n ≠ 0)
   
   - $c_{mn} = \int_{-1}^{1} \int_{-1}^{1} f(x', y') \sin(m\pi x') \cos(n\pi y') \, dx' \, dy'$ (если m ≠ 0)
   
   - $d_{mn} = \int_{-1}^{1} \int_{-1}^{1} f(x', y') \sin(m\pi x') \sin(n\pi y') \, dx' \, dy'$ (если m ≠ 0 и n ≠ 0)

   Численно:
   $$a_{mn} = \sum_{i=1}^{N_x} \sum_{j=1}^{N_y} f(x'_i, y'_j) \cos(m\pi x'_i) \cos(n\pi y'_j) \cdot dA$$

   Аналогично для остальных коэффициентов.

## Ортонормированность базиса

Базисные функции ортонормированы на [-1, 1] × [-1, 1]:
$$\int_{-1}^{1} \int_{-1}^{1} \cos(m\pi x) \cos(n\pi y) \cos(m'\pi x) \cos(n'\pi y) \, dx \, dy = \delta_{mm'} \delta_{nn'}$$

Поэтому коэффициенты дают точное представление при достаточном числе мод.

## Параметры класса

- `order`: Максимальный порядок гармоник
- `x_order`, `y_order`: Максимальные порядки по x и y

## Применение

Фурье-разложение используется в обработке изображений, анализе сигналов, решении дифференциальных уравнений и фильтрации частот.
